# Pizza
Гайд как сделать все чотка
1. скачать зип файл
2. разорхировать
3. открыть в редакторе
4. написать в консоле git bash "python -m venv venv"
5. активировать виртуальное окружение "source venv/scripts/activate"
6. затем скачать зависимости "pip install -r requirements.txt"
7. потом перейти в директорию pizza с помощью команды "cd pizza"
8. после чего прописать 'python manage.py runserver'
